# LinuxDojo 可视化功能增强

## 🎯 项目概述

LinuxDojo 是一个交互式的 Linux 命令学习平台，现已增强了强大的可视化功能，特别针对网络命令和防火墙管理提供了直观的动画演示和实时模拟。

## 🚀 新增功能

### 1. 🔥 iptables 防火墙可视化器

#### 核心功能
- **实时防火墙规则管理** - 可视化添加、删除、修改 iptables 规则
- **数据包流动模拟** - 动画展示数据包在防火墙规则链中的处理过程
- **网络拓扑显示** - 展示网络接口（eth0、lo、eth1）和连接关系
- **多表支持** - 支持 filter、nat、mangle 表的切换和管理
- **实时日志记录** - 详细记录所有操作和规则匹配过程

#### 交互特性
- 📦 **数据包模拟器**：自定义源IP、目标IP、协议、端口
- ⚡ **快速规则添加**：一键添加常用规则（阻止IP、允许SSH、允许HTTP）
- 🎭 **规则链可视化**：INPUT、FORWARD、OUTPUT 三个链的实时状态
- 🔄 **动态匹配过程**：高亮显示规则匹配过程，包括成功、失败、处理中状态

#### 使用场景
```bash
# 当用户输入以下命令时，自动启用 iptables 可视化器
iptables -L
iptables -A INPUT -p tcp --dport 22 -j ACCEPT
iptables -I INPUT -s 192.168.1.100 -j DROP
```

### 2. 🌐 网络命令可视化器

#### 支持的命令

##### 🏓 ping 命令可视化
- **实时统计面板**：发送/接收/丢包率/平均延迟
- **网络路径动画**：本机到目标主机的数据包飞行动画
- **连接状态指示**：实时显示网络连通性
- **详细日志输出**：完整的 ping 响应日志

##### 📊 netstat 命令可视化
- **连接状态分布**：ESTABLISHED、LISTEN、TIME_WAIT 等状态的统计图表
- **协议分布图表**：TCP、UDP、TCP6、UDP6 的比例展示
- **活跃连接表格**：详细的连接信息列表
- **实时数据更新**：动态刷新网络连接状态

##### 🛤️ traceroute 命令可视化
- **逐跳路由显示**：每个路由节点的详细信息
- **延迟时间展示**：每跳的三次测试延迟
- **动态追踪过程**：实时显示路由追踪进度
- **网络拓扑映射**：可视化网络路径结构

#### 使用场景
```bash
# 以下命令会自动启用相应的网络可视化器
ping google.com
ping -c 4 8.8.8.8
netstat -tuln
netstat -an
traceroute google.com
traceroute 8.8.8.8
```

## 🔧 安装和使用

### 1. 环境要求
```bash
Node.js >= 16.0.0
npm >= 8.0.0
```

### 2. 安装依赖
```bash
cd linuxdojo
npm install
```

### 3. 启动开发服务器
```bash
npm run dev
```

### 4. 访问测试页面
```
http://localhost:5173
```

## 🧪 测试功能
