# LinuxDojo 可视化功能演示指南

## 🚀 快速开始

### 启动项目

```bash
cd linuxdojo
npm install
npm run dev
```

### 访问地址
```
http://localhost:5173
```

## 🎯 核心演示功能

### 1. iptables 防火墙可视化演示

#### 场景一：查看防火墙规则
1. 在命令输入区输入：`iptables -L`
2. 观察：
   - 三个规则链的可视化界面（INPUT、FORWARD、OUTPUT）
   - 当前表选择器（filter/nat/mangle）
   - 网络拓扑图显示

#### 场景二：数据包流动模拟
1. 在数据包模拟器中设置：
   - 源IP：`192.168.1.100`
   - 目标IP：`10.0.0.1`
   - 协议：`tcp`
   - 端口：`80`
2. 点击"🚀 发送数据包"
3. 观察：
   - 数据包在网络拓扑中的流动动画
   - 规则链中规则的匹配过程（黄色=处理中，绿色=匹配成功）
   - 实时日志记录匹配结果

#### 场景三：快速规则管理
1. 使用快速规则按钮：
   - "🚫 阻止IP访问" - 添加阻止特定IP的规则
   - "✅ 允许SSH" - 添加允许SSH连接的规则
   - "🌐 允许HTTP" - 添加允许HTTP访问的规则
2. 观察规则链中新增的规则
3. 点击规则旁的"❌"按钮删除规则

#### 场景四：自定义规则添加
1. 点击任意规则链的"➕ 添加规则"
2. 在模态框中配置：
   - 协议：选择 TCP/UDP/ICMP
   - 源IP/目标IP：设置IP地址或网段
   - 端口：设置源端口和目标端口
   - 动作：选择 ACCEPT/DROP/REJECT
3. 观察新规则的添加和日志记录

### 2. 网络命令可视化演示

#### ping 命令演示
1. 输入命令：`ping google.com`
2. 观察：
   - 实时统计面板（发送/接收/丢包率/平均延迟）
   - 网络路径中的数据包飞行动画
   - 详细的ping响应日志

#### netstat 命令演示
1. 输入命令：`netstat -tuln`
2. 观察：
   - 连接状态统计（ESTABLISHED、LISTEN、TIME_WAIT等）
   - 协议分布图表（TCP、UDP、TCP6、UDP6）
   - 活跃连接详细列表

#### traceroute 命令演示
1. 输入命令：`traceroute google.com`
2. 点击"🚀 开始追踪"
3. 观察：
   - 逐跳路由节点的动态显示
   - 每跳的延迟时间测试
   - 路由追踪进度动画
